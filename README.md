# Tauri + Next.js + Typescript

This template should help get you started developing with Tauri, Next.js and Typescript.

## Get Started

1. Install packages

```sh
pnpm i
```

2. Run

```sh
pnpm tauri dev
```

## Release

1. Change the bundle identifier in `tauri.conf.json > tauri > bundle > identifier`.
1. Set new tag
